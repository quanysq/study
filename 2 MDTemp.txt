***问题***

1. nvm 安装 nodejs 运行报错：exit status 1: 'D:\Program' 乱码
	原因：nvm安装到了有空格的路径上（D:\Program Files），导致切换失败
	Link: https://blog.csdn.net/mrhaoxiaojun/article/details/122505142
2. 在vscode终端运行 npm run dev 出错：npm : 无法加载文件 D:\Learn\NodeJs\nvm\npm\npm.ps1。未对文件 D:\Learn\NodeJs\nvm\npm\npm.ps1 进行数字签名。无法在当前系统上运行该脚本。有关运行脚本和设置执行策略的详细信息，请参阅 https:/go.microsoft.com/fwlink/?LinkID=135170 中的 about_Execution_Policies。
	解决：直接删除 D:\Learn\NodeJs\nvm\npm\npm.ps1，或者把它改名
	Link：http://www.caotama.com/1424076.html
3. Win10环境变量Path的开头处增加Java变量后，编辑多行变成单行？
	把 C:\WINDOWS\system32; 加到最前
	https://blog.csdn.net/m0_67297673/article/details/123843510
4. 网上邻居看得见局域网机器，打开时提示找不到网络路径
	原因：该机器是双网卡，一个IP在局域网内，另一个IP不在局域网内
	解决：TrueNas 删除其中物理卡(re0)的IP配置
5. 打开网上邻居，提示：网络发现已关闭,网络计算机和设备不可见
	控制面板 - 网络和Internet - 网络和共享中心 - 更改高级共享设置 - 启用网络发现
	https://g.pconline.com.cn/x/1553/15536456.html
